# Java Calculator

This is a simple calculator program built using Java.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Modulus
- Continuous operation (runs until user exits)

## How it works
1. User selects an operation from the menu
2. User enters two numbers
3. Program performs the calculation
4. Result is displayed

## Concepts Used
- Loops (while loop)
- Switch case
- User input (Scanner)
- Basic error handling
